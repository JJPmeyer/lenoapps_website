# Introduction

**Data Bridge** helps you bring data into monday.com fast, cleanly, and reliably.

Set up your data flow once, map your fields, and run repeatable loads without manual copy-paste. Data Bridge also supports complex structures, so you can load parent items, subitems, and multi-level subitems that match the hierarchy in your source data.

## Key Benefits

- Move large data sets in minutes instead of hours of manual work
- Map source fields to monday.com columns with a simple setup step
- Run scheduled loads automatically with less manual intervention
- Load data to standard boards and multi subitems boards from one workflow
- Multi subitems board support is a key differentiator, and no other app currently offers this level of support
- Monitor clear progress logs while runs are in progress
- Access a growing set of source connectors, with the option to request any source your team needs

## App Screens

Data Bridge has two main screens:

- **Import Setup:** Configure source, board target, hierarchy options, and field mapping
- **Template Manager:** Save and reuse templates, run loads, edit templates, and manage schedules

## Source Coverage

- Data Bridge supports multiple source types through one workflow
- New sources are added continuously
- Need a specific source? Request it at **support@lenoapps.com**

## Recommended Workflow

1. Connect your source
2. Map your fields to the right monday.com columns
3. Run data loads, monitor logs, and manage templates and schedules for repeat runs
