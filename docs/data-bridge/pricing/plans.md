# Pricing Plans

Data Bridge offers flexible plans for individuals and teams that need reliable data loading into monday.com.

For the most up-to-date plan names, limits, and prices, please check the Data Bridge listing on the monday.com marketplace.

## What every plan is designed for

- Faster data loading from multiple source connectors
- Field mapping from source data to monday.com columns
- Scheduled runs for repeat, hands-free data loading
- Standard boards and multi subitems boards support in one workflow
- Multi subitems board support is a key differentiator, and no other app currently offers this level of support
- Import logs and run controls for visibility
- Ongoing source connector expansion with source-request support

## Billing notes

- Billing is handled through monday.com marketplace billing
- Plan upgrades and downgrades are managed from your monday.com account
