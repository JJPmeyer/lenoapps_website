# Template Manager

## Step 1: Create a template

1. Go to **Import setup**
2. Configure source, board, fields, and mapping
3. Enter a **Template name**
4. Click **Save template**

## Step 2: Open Template Manager

1. Click **Template manager**
2. Review your saved templates in the table with status and actions

## Step 3: Run a template load

1. In the template row, click **Run import**
2. The logs section opens below the table
3. Track progress until completion

## Step 4: Stop a running load

1. If a template load is running, click **Stop**
2. The job is canceled and status updates after refresh

## Step 5: Edit a template

1. Click **More actions** on a row
2. Select **Edit template**
3. The template editor opens below the table
4. Update settings and click **Save template**
5. Use **Cancel** to close without saving

## Step 6: Edit schedule (if enabled)

1. Click **Set schedule** / schedule action
2. Update frequency/time settings
3. Save schedule changes

Saving a schedule does not run a load.

Scheduling is ideal when you want repeat loads without manual reruns.

## Step 7: Delete a template

1. Click **More actions**
2. Select **Delete template**
3. Confirm deletion

## Step 8: Refresh template data

1. Click **Refresh** to load latest statuses and results

## Notes for users

- If one template load is running, other row actions may be disabled
- Some connector types may require reconnecting source details when running or editing
- Saving edits updates the template only; loads run only when you click **Run import**
